# homebrew-axon

Homebrew tap for [axon](https://github.com/rudraptpsingh/axon) -- local hardware intelligence for AI coding agents.

## Install

```bash
brew tap rudraptpsingh/axon
brew install axon
```

## After installing

Configure your agents, then restart them:

```bash
axon setup           # configures all detected agents
axon diagnose        # quick smoke test
```

## Updating

```bash
brew update
brew upgrade axon
```
