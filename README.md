# homebrew-axon

Homebrew tap for [axon](https://github.com/rudraptpsingh/axon) -- local hardware intelligence for AI coding agents.

## Install

```bash
brew tap rudraptpsingh/axon
brew install axon
```

## After installing

Restart your AI agent (Claude Desktop, Cursor, VS Code). axon auto-configures itself on first run.

```bash
axon diagnose    # Quick smoke test
```

## Updating

```bash
brew update
brew upgrade axon
```
